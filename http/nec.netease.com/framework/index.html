    重置样式，清除浏览器默认样式，并配置适合设计的基础样式

    功能样式，从常用样式方法中抽离，按需使用

    媒体查询，为不同的设备和大小配置不同的样式

    常见动画效果的集合，主要用于效果演示和参考，基于css3

    包含完整头部信息和主体结构的HTML模板
