插件

帮助你快速有效的使用代码库（布局、模块、元件）。支持多种常用流行的前端开发软件：Eclipse、Vim、Sublime Text2……

  • Eclipse插件

    xml文件，通过软件自带的Snippets视图导入，可视化的代码片段插入

  • Eclipse插件2

    xml文件，通过软件自带的模板功能导入，自动代码提示

  • Vim插件

    snippets文件，通过结合其他插件实现，自动代码提示

  • Sublime插件

    snippets压缩文件，通过软件自带的Snippets工具实现，自动代码提示
